# LLM Prompt Injection Defences 

Seminar paper for the course **Expertise & Communication**
THWS Würzburg-Schweinfurt SS 2026

**Authors** \
Krüger Denis \
Mautner Sebastian

---

## Topic 
This paper analyzes **Prompt Injection attacks** in Large Language Model (LLM)m systems and evaluates current defence machanisms.

The focus lies on:

- the underlying vulnerability caused by mixing instructions and data 
- structured query approaches such as **StruQ**
- detection-based and system-level defences
- evaluation using recent benchmarks 
- limitations of current protection mechanisms

---

## Research Question 
> Wie wirksam ist die Trennung von Instruktionen und Daten als Verteidgungsstrategie gegen Prompt Injection, und wo liegen die Grenzen aktueller LLM Prompt Injection Defences?
