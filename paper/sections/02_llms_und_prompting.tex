\section{LLMs and Prompting}
\label{sec:llms}

Large Language Models are pretrained to predict the next syllable/ word-piece (token) from preceding context.
After instruction tuning, this enables tasks such as summarisation, translation,
question answering, and code generation
\parencites{brown2020language}{chen2025struq}.

In an LLM-integrated application, the model context may contain
application-provided instructions, a user request, retrieved data, and tool
results. Role markers and delimiters can signal that some components should
receive higher priority than others, and instruction-hierarchy training can
strengthen this behaviour
\parencite{wallace2024instruction}. However, these distinctions remain inputs
interpreted by the model rather than an independently enforced authority
mechanism.
